# Система управления рестораном

## Описание

Админ панель для управления рестораном, построенная на Next.js и Convex. Система позволяет управлять категориями блюд и позициями меню с возможностью загрузки изображений.

## Функциональность

### Категории
- ✅ Создание новых категорий
- ✅ Редактирование существующих категорий
- ✅ Удаление категорий (с проверкой на наличие позиций меню)
- ✅ Просмотр списка всех категорий

### Позиции меню
- ✅ Создание новых позиций меню
- ✅ Редактирование существующих позиций
- ✅ Удаление позиций меню
- ✅ Загрузка изображений через ImageBan API
- ✅ Просмотр списка всех позиций с информацией о категориях
- ✅ Поддержка скидок

## Технический стек

- **Frontend**: Next.js 14, React, TypeScript, Tailwind CSS
- **Backend**: Convex (база данных, API, real-time обновления)
- **Хостинг изображений**: ImageBan API

## Настройка

### 1. Установка зависимостей

```bash
npm install
```

### 2. Настройка Convex

1. Создайте проект на [convex.dev](https://convex.dev)
2. Скопируйте URL вашего проекта
3. Добавьте URL в переменные окружения

### 3. Настройка ImageBan API

1. Зарегистрируйтесь на [imageban.ru](https://imageban.ru)
2. Перейдите в профиль и получите CLIENT_ID
3. Создайте файл `.env.local` в корне проекта:

```env
# ImageBan API Configuration
NEXT_PUBLIC_IMAGEBAN_CLIENT_ID=ваш_client_id_здесь

# Convex Configuration
NEXT_PUBLIC_CONVEX_URL=ваш_convex_url_здесь
```

### 4. Запуск проекта

```bash
npm run dev
```

## Структура базы данных

### Таблица `categories`
- `_id` - уникальный идентификатор
- `name` - название категории

### Таблица `menuPositions`
- `_id` - уникальный идентификатор
- `name` - наименование блюда
- `categoryId` - ссылка на категорию
- `articleNumber` - артикул (число)
- `price` - цена (число)
- `photo` - URL изображения (опционально)
- `weight` - граммовка (число)
- `structure` - состав блюда
- `discountPrice` - цена со скидкой (опционально)

## API функции

### Категории
- `createCategory` - создание категории
- `listCategories` - получение списка категорий
- `getCategory` - получение категории по ID
- `updateCategory` - обновление категории
- `deleteCategory` - удаление категории

### Позиции меню
- `createMenuPosition` - создание позиции меню
- `listMenuPositions` - получение списка позиций с информацией о категориях
- `listMenuPositionsByCategory` - получение позиций по категории
- `getMenuPosition` - получение позиции по ID
- `updateMenuPosition` - обновление позиции
- `deleteMenuPosition` - удаление позиции

## Загрузка изображений

Система использует ImageBan API для загрузки изображений:

1. Пользователь выбирает файл изображения
2. Файл отправляется на ImageBan API
3. Полученный URL сохраняется в базе данных
4. Изображение отображается в интерфейсе

### Поддерживаемые форматы
- JPEG, JPG
- PNG
- GIF

### Ограничения
- Максимальный размер файла: 10 МБ
- Лимит загрузок: 1000 в сутки для бесплатных аккаунтов

## Разработка

1. Создайте новую таблицу в `convex/schema.ts`
2. Добавьте функции в `convex/restaurant.ts`
3. Создайте компонент в `components/`
4. Добавьте навигацию в `app/page.tsx`
