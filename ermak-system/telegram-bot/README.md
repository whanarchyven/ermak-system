# Telegram Bot для Ermak System

## Описание
Telegram бот для системы Ermak ERP, который позволяет:
- Регистрировать пользователей
- Сохранять сообщения в базу данных
- Показывать статистику
- Управлять настройками пользователей

## Установка и запуск

### 1. Установка зависимостей
```bash
npm install
```

### 2. Настройка переменных окружения
Создайте файл `.env.local` в корне проекта со следующими переменными:
```env
NEXT_PUBLIC_CONVEX_URL=https://your-deployment.convex.cloud
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
```

### 3. Запуск Convex функций
```bash
npm run dev:backend
```

### 4. Запуск бота
```bash
# Обычный запуск
npm run bot

# Запуск в режиме разработки (с автоперезагрузкой)
npm run bot:dev
```

## Команды бота

### Основные команды:
- `/start` - Начать работу с ботом
- `/help` - Показать список команд
- `/profile` - Просмотр профиля
- `/stats` - Статистика сообщений
- `/settings` - Настройки уведомлений

### Административные команды (только для админов):
- `/admin` - Панель администратора

## Архитектура

### База данных (Convex):
- `clients` - Пользователи бота
- `messages` - Сообщения пользователей
- `logs` - Логи системы

### Файлы:
- `bot.ts` - Основной файл бота
- `types.ts` - TypeScript типы
- `README.md` - Документация

## Разработка

### Добавление новых команд:
1. Добавьте обработчик в `commandHandlers`
2. Создайте функцию-обработчик
3. Обновите справку в `/help`

### Добавление новых типов сообщений:
1. Обновите схему в `convex/schema.ts`
2. Добавьте обработку в `bot.ts`
3. Обновите типы в `types.ts`
